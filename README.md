# TrollBot 🤬

**TrollBot** is your ultimate Discord companion for trolling your friends, spicing up conversations with epic humor, and unleashing your inner trickster. Whether you're looking to drop some cheeky insults, consult a sarcastic magic 8-ball, or mock someone's spelling mistakes, **TrollBot** has got your back with hilarious commands that will leave everyone laughing—or maybe crying!

- **Spelling Police**: Catch those minor spelling mistakes with GIF reactions

- **Insult Generator**: Playfully roast someone with witty one-liners

## 🚀 Installation

Follow these steps to get TrollBot up and running:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-repo-name/trollbot.git
   cd trollbot
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env` file in the root directory and add the following:
   ```env
   DISCORD_TOKEN=your_discord_token
   CLIENT_ID=your_client_id
   TENOR_API_KEY=your_tenor_api_key
   ```

4. **Run the bot:**
   ```bash
   node index.js
   ```

## 📚 Commands

### 🎯 Core Commands
- `/bad_spel` - Call out spelling mistakes with a perfect GIF reaction
- `/swear` - Get a random swear word when you need to express yourself
- `/rage` - Express your fury with epic rage responses
- `/destroy` - Unleash virtual destruction with style
- `/mock` - Convert the last message into mocking aLtErNaTiNg CaSe

### 🎲 Interactive Commands
- `/insult @user` - Roast someone with a witty insult (special protection for Armaan!)
- `/8ball [question]` - Ask the magic 8-ball your burning questions
- `/funfact` - Get a random useless fact to expand your knowledge

### 🎭 Special Features
- **GIF Integration**: Spelling mistake reactions come with perfectly timed GIFs via Tenor API
- **Random Responses**: Every command delivers variety with randomized responses
